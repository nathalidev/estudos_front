Modelo de caixa

O modelo de caixa do CSS descreve a estrutura de cada elemento HTML como uma caixa retangular composta por quatro áreas: conteúdo, preenchimento (padding), borda (border) e margem (margin).

Componentes do modelo de caixa

  • Conteúdo: A área onde o texto e as imagens aparecem.
  • Preenchimento (Padding): O espaço entre o conteúdo e a borda.
  • Borda (Border): A linha que envolve o preenchimento e o conteúdo.
  • Margem (Margin): O espaço fora da borda que separa o elemento de outros elementos.
